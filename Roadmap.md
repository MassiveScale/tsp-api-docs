# Feature Roadmap

**Optional Base URL**
An `base-url` option that, when procided, should be used when rendering HTTP request paths. The path may be absolute or reletive.

**Type Linking**
When rendering a reference type name for a property type, request body type, and return type create a link to that resource documentation page

**External Standards**
When referencing an RFC, IEE, or ISO standard link to it's external definition

**Resource Map**
Render map of resources and relationships as a Mermaid chart
